# Consolidates thin Actions Adapters, and Use Cases

An agent skill for merging Actions Adapters and Use Cases after an initial refactoring pass.

This repo contains a workflow skill that performs the next-step consolidation after a first-pass refactor. It identifies related action handlers, adapter logic, and use case boundaries, then helps merge or reorganize them into a cleaner structure with less duplication, clearer ownership, and fewer needless abstraction layers.
